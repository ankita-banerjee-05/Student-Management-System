#  Student Management System

##  About

This is a simple Student Management System built using Python and MySQL. It allows basic operations like adding, viewing, updating, and deleting student records. The project helped me understand how backend logic connects with a database.

##  Features

* Add student details
* View all students
* Search by ID
* Update student records
* Delete student records

##  Tech Stack

* Python
* MySQL
* mysql-connector

##  How to Run

1. Install dependency:
   pip install mysql-connector-python
2. Create database: `student_db`
3. Run:
   python student_management.py

##  Learning Outcome

* Learned database connectivity with Python
* Implemented CRUD operations
* Improved backend logic and problem-solving
